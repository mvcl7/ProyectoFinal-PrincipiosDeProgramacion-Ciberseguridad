## Requisitos previos
- Tener Python instalado → https://www.python.org/downloads/
- Al momento de instalar Python en Windows, tildar la opción **"Add Python to PATH"**

---

## Configuración inicial (hacerlo UNA sola vez)

1. Clonar o descargar el proyecto
2. Abrir una terminal en la carpeta del proyecto
3. Ejecutar:

```bash
python setup.py
```

> Si usás Windows y nmap pide instalación manual, seguí las instrucciones que aparecen en pantalla, luego volvé a correr `python setup.py` para verificar.

---

## Cómo usar el proyecto

Una vez completado el setup, ejecutar:

```bash
python main.py
```
